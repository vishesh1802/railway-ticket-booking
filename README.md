# Railway Ticket Booking

A ticket-booking system for railway reservations built in Java Swing Framework.  
This repository implements core booking flows such as searching trains, selecting seats, reserving tickets, and cancellation.

## Features

- Search for available trains between two stations  
- Display seat availability by class (e.g., Sleeper, AC)  
- Book a ticket with passenger details, date, class, etc  
- View booked tickets and cancel if needed  
- Simple text-based (or console) UI for demonstration
